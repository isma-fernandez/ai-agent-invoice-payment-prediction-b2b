B2B Invoice Payment Prediction Agent
=====================================

Sistema multiagente para predicción de pagos de facturas B2B.

Arquitectura
------------

El proyecto está organizado en tres capas principales:

- **apps/**: Aplicaciones (orchestrator, agents, frontend)
- **shared/**: Código compartido (clients, data, models, utils)
- **services/**: Servicios MCP (prediction, memory)
